# Contador Clientes Piscina Municipal Sobreposta 2021

Este projeto tem como objetico a criação de uma plataforma que contabiliza os clientes que se encontram no estabelecimento.
Foi desenvolvido no âmbito pessoal pelos progamadores João Antunes e Ricardo Guimarães.
--------------------------------------------------------------------------
# Método de Utilização 
O responsável pela bilheteira irá realizar login na página de admin da plataforma, onde terá acesso ao botão de incrementar e subtrair, que são referentes às pessoas que entram e saem das instalações da Piscina Municipal de Sobreposta.
O utilizador que procura saber a lotação do espaço irá abrir o link da plataforma onde será mostrado o número de pessoas que se encontram no estabelecimento assim como uma bandeira de lotação, baseada num sistema de semáforo, verde, amarelo e vermelho.
